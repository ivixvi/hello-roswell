# Hello-Roswell

## Usage

1. `path/to/hello-roswell`
1. `./script/{script-name}.ros`
    - for windows `ros ./script/{script-name}.ros`

### `script-name` list
- `hello-world`
    - output "Hello, World"
- `yubaba`
    - Yubaba takes away your name. 
    - inspired by [Javaで湯婆婆を実装してみる](https://qiita.com/Nemesis/items/c7192a7c510788d2cba2)
